Especificidade no CSS

Cada elemento, classe, id, vale pontos que são usados pelos navegadores na hora de renderizar os estilos

A pontuação é feita da seguinte maneira:

  • Composta por 4 números (0000)
  • Quanto mais a esquerda, meior a relevância da pontuação
  • Vamos chamr esses números de (a,b,c,d)
  • Regra da pontuação(sobrescrevendo a herança):
    • Cada elemento / pseudo elemento -> d = +1. (0,0,0,1)
    • Cada classe /pseudo classe /atributo -> c = +1 (0,0,1,0)
    • Cada id -> b = +1 (0,1,0,0)
    • Estilo inline -> a = +1 (1,0,0,0)

Exemplo de pontuação:

div#container ul li -> (0,1,0,3)

Quando ocorre um empate, o ultimo seletor adicionado será o de desempate
